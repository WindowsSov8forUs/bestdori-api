# 简介

**Bestdori API** 是一个用 Python 编写的调用 [Bestdori](https://bestdori.com/) 各种 API 与资源下载的库，大致包括了社区帖子的处理以及各种 [BanG Dream！少女乐团派对](https://zh.moegirl.org.cn/BanG_Dream!_%E5%B0%91%E5%A5%B3%E4%B9%90%E5%9B%A2%E6%B4%BE%E5%AF%B9%EF%BC%81) 游戏内资源的获取。

::: warning
此项目目前已趋于完善，但仍然不排除存在错误的可能性，如遇到问题请尽快[反馈](https://github.com/WindowsSov8forUs/bestdori-api/issues)。
:::

::: info bestdori-api 全面支持同步与异步
异步使用方法与同步类似，可几乎[**无缝切换**](/fast-start/#async-sync)
:::
